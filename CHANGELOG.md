# Changelog

All notable changes to Spectator are documented here.

## [0.1.0] - 2026-07-11

First public release.

- Live, read-only Hermes Desktop session viewing through a loopback-only tap.
- Streamed replies, safe tool activity, approval state, viewer count, and reconnect recovery.
- Mandatory secret redaction before browser fan-out or persistence.
- Session-scoped chronological history and accessible light/dark themes.
- Windows managed startup, deterministic teardown, and standalone Hermes recovery.
- Cloudflare Quick Tunnel documentation for temporary public links.
